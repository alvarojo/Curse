These are the functions which can be called on a minecraft:player_killed_entity criteria 
trigger.



getEntityData:
    Notes:
        Returns the EntityData object so functions can be called on it.

setEntityType:
    Arguments:
        String
    Usage:
        entity type
    Notes:
        Sets the entity type.

getDamageSourceData:
    Notes:
        Returns the DamageSourceData object so functions can be called on it.
